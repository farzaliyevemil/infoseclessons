# CompTIA Sertifikatları

Bu bələdçi **23 mart 2026** tarixinə geniş istifadə olunan **CompTIA sertifikatlarını** əhatə edir. CompTIA portfelində branding, renewal və struktur dəyişə bildiyi üçün son vəziyyəti rəsmi CompTIA saytından yoxlamaq lazımdır.

---

## 🧭 CompTIA karyerada necə istifadə olunur?

CompTIA daha çox bunlar üçün faydalıdır:

- vendor-neutral fundamental baza
- erkən karyera üçün aydın roadmap
- support, networking, cloud və cybersecurity üzrə praktik sertifikatlar

Vendor-specific yollardan fərqli olaraq, CompTIA çox vaxt geniş baza qurmaq üçün istifadə olunur.

---

## 📚 Əsas sertifikat qrupları

### Entry / Support

| Sertifikat | Fokus |
| --- | --- |
| **Tech+** | Başlanğıc texnologiya əsasları |
| **A+** | End-user support, hardware, software, troubleshooting |

### Infrastructure / Operations

| Sertifikat | Fokus |
| --- | --- |
| **Network+** | Şəbəkə əsasları, əməliyyat, troubleshooting |
| **Server+** | Server hardware, administrasiya, əməliyyat |
| **Linux+** | Linux administrasiyası və troubleshooting |
| **Cloud+** | Multi-cloud əməliyyatları, deployment və təhlükəsizlik |

### Cybersecurity

| Sertifikat | Fokus |
| --- | --- |
| **Security+** | Əsas təhlükəsizlik bazası |
| **CySA+** | Detection, analysis, incident response, defensive əməliyyatlar |
| **PenTest+** | Praktik penetration testing və assessment |
| **SecurityX** | Expert-level security architecture və engineering |

### Data / Project

| Sertifikat | Fokus |
| --- | --- |
| **Data+** | Data analysis əsasları |
| **Project+** | Layihə koordinasiyası və delivery əsasları |

---

## 🔐 Vacib 2025+ yeniliyi

CompTIA **SecurityX** artıq köhnə **CASP+** adını əvəz edən cari advanced cybersecurity sertifikatıdır. Köhnə roadmap-lərdə CASP+ görsən, onu cari deyil, əvvəlki ad kimi qəbul et.

---

## 🛣️ Sadə karyera yolları

### Helpdesk / IT Support

`A+` -> `Network+` -> `Security+`

### Sistem və İnfrastruktur

`A+` -> `Network+` -> `Server+` və ya `Linux+` -> `Cloud+`

### Cybersecurity

`A+` və ya `Network+` -> `Security+` -> `CySA+` və ya `PenTest+` -> `SecurityX`

### Cloud / Platform Operations

`Network+` -> `Linux+` və ya `Server+` -> `Cloud+`

---

## 🧱 Stackable model niyə önəmlidir?

CompTIA-nın güclü tərəflərindən biri **stackable certification** modelidir.

Cari pathway strukturundan nümunələr:

- `A+` + `Network+` -> **CompTIA IT Operations Specialist**
- `A+` + `Network+` + `Security+` -> **CompTIA Secure Infrastructure Specialist**
- `Security+` + `CySA+` -> **CompTIA Security Analytics Professional**
- `Security+` + `PenTest+` + `CySA+` -> **CompTIA Network Security Professional**

Bu da CompTIA-nı tək-tək exam-lərdən ibarət yox, progression framework kimi dəyərli edir.

---

## 🔄 Renewal əsasları

CompTIA bir çox sertifikat üçün **Continuing Education (CE)** modelindən istifadə edir.

Vacib məqamlar:

- Daha yüksək sertifikat bəzi aşağı sertifikatları yeniləyə bilir
- `SecurityX` aşağı səviyyədə `CySA+`, `PenTest+`, `Security+`, `Network+`, `A+` kimi sertifikatları renew edə bilir
- `Security+` -> `Network+` və `A+`
- `Network+` -> `A+`

Bu da uzunmüddətli inkişaf planını rahatlaşdırır.

## ✅ Haradan başlamaq olar?

**Əgər IT-yə yenisənsə:**
`A+`

**Əgər support-dasan və networking istiqamətinə keçmək istəyirsənsə:**
`Network+`

**Əgər cybersecurity istəyirsənsə, amma baza da lazımdırsa:**
`Security+`

**Əgər blue team / SOC istiqaməti istəyirsənsə:**
`Security+` -> `CySA+`

**Əgər offensive security istiqamətinə getmək istəyirsənsə:**
`Security+` -> `PenTest+`

`SecurityX` istiqamətinə get
