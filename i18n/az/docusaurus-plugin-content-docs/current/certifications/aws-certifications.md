# AWS Sertifikatları

Bu bələdçi **23 mart 2026** tarixinə əsas **AWS sertifikatlarını** əhatə edir. AWS exam adlarını və portfel strukturunu zaman-zaman dəyişdirdiyi üçün imtahan planlamazdan əvvəl son vəziyyəti rəsmi AWS Certification səhifəsində yoxlamaq lazımdır.

---

## 🧭 AWS sertifikatları necə qurulub?

AWS sertifikatları səviyyələrə bölür:

| Səviyyə | Məqsəd |
| --- | --- |
| **Foundational** | Geniş AWS biliyi və cloud əsasları |
| **Associate** | Rol əsaslı texniki bacarıqlar |
| **Professional** | Daha dərin arxitektura, əməliyyat və engineering |
| **Specialty** | Dar sahədə yüksək səviyyəli ekspertiza |

---

## ☁️ Foundational

| Sertifikat | İmtahan | Fokus |
| --- | --- | --- |
| AWS Certified Cloud Practitioner | CLF-C02 | Ümumi AWS Cloud biliyi |
| AWS Certified AI Practitioner | AIF-C01 | Əsas AI, ML və generative AI anlayışları |

Bunlar daha uyğundur:

- AWS-ə yeni başlayanlar
- texniki olmayan stakeholder-lər
- dərin texniki yola keçməzdən əvvəl cloud literacy istəyənlər

---

## 🧰 Associate

| Sertifikat | İmtahan | Fokus |
| --- | --- | --- |
| AWS Certified Solutions Architect - Associate | SAA-C03 | AWS həllərinin dizaynı |
| AWS Certified Developer - Associate | DVA-C02 | AWS tətbiqlərinin hazırlanması və saxlanması |
| AWS Certified CloudOps Engineer - Associate | SOA-C03 | Deployment, idarəetmə və əməliyyatlar |
| AWS Certified Data Engineer - Associate | DEA-C01 | Data pipeline və data store-lar |
| AWS Certified Machine Learning Engineer - Associate | MLA-C01 | ML həllərinin qurulması və saxlanması |

> Köhnə materiallarda operations associate yolu hələ də **SysOps Administrator - Associate** kimi görünə bilər. Cari ad **CloudOps Engineer - Associate**-dir.

---

## 🏗️ Professional

| Sertifikat | İmtahan | Fokus |
| --- | --- | --- |
| AWS Certified Solutions Architect - Professional | SAP-C02 | Advanced architecture design |
| AWS Certified DevOps Engineer - Professional | DOP-C02 | Delivery, automation və operations |
| AWS Certified Generative AI Developer - Professional | AIP-C01 | AWS üzərində advanced AI həllərinin dizaynı və deployment-u |

Bu səviyyə ilk AWS imtahanı üçün yox, daha təcrübəli practitioner-lər üçündür.

---

## 🎯 Specialty

| Sertifikat | İmtahan | Fokus |
| --- | --- | --- |
| AWS Certified Advanced Networking - Specialty | ANS-C01 | Mürəkkəb AWS və hybrid şəbəkə arxitekturası |
| AWS Certified Security - Specialty | SCS-C03 | AWS workload və məlumat təhlükəsizliyi |
| AWS Certified Machine Learning - Specialty | MLS-C01 | Köhnə ML specialty istiqaməti |

Vacib qeyd:

- AWS sənədlərində **AWS Certified Machine Learning - Specialty** üçün **31 mart 2026** son imtahan tarixi göstərilir

---

## 🛣️ Sadə karyera yolları

### Başlanğıc / ümumi cloud

`Cloud Practitioner` -> `Solutions Architect Associate`

### Cloud administration / operations

`Cloud Practitioner` -> `CloudOps Engineer Associate` -> `DevOps Engineer Professional`

### Development

`Cloud Practitioner` -> `Developer Associate` -> `DevOps Engineer Professional`

### Architecture

`Cloud Practitioner` -> `Solutions Architect Associate` -> `Solutions Architect Professional`

### Cloud security

`Cloud Practitioner` -> `Solutions Architect Associate` və ya `CloudOps Engineer Associate` -> `Security Specialty`

---

## ✅ Haradan başlamaq daha düzgündür?

**Əgər AWS-ə tam yenisənsə:**
`AWS Certified Cloud Practitioner`

**Əgər ən çevik texniki başlanğıc istəyirsənsə:**
`Solutions Architect - Associate`

**Əgər operations/admin yönümlüsənsə:**
`CloudOps Engineer - Associate`

**Əgər developer-sənsə:**
`Developer - Associate`

**Əgər security istiqaməti istəyirsənsə:**
Əvvəl associate səviyyəsində güclü AWS bazası qur, sonra `Security - Specialty`-ə keç
