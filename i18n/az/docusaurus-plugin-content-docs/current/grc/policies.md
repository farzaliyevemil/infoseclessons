# Təhlükəsizlik Siyasətləri

Bu səhifə gələcəkdə təhlükəsizlik siyasətlərinin yazılması və saxlanması üçün praktik bələdçiyə çevriləcək:

- Siyasətin məqsədi və əhatə dairəsi
- Rol və məsuliyyətlər
- Review və approval prosesi
- İT və təhlükəsizlik komandaları üçün əsas siyasət ailələri

Tam dərs hələ hazırlanır.
