# Initial Access

Bu səhifə gələcəkdə adversary simulation və security assessment zamanı görünən initial access yollarını əhatə edəcək:

- Phishing və social engineering
- Password attack və exposed credential ssenariləri
- Public-facing application zəiflikləri
- Valid account və misconfiguration nümunələri

Tam dərs hələ hazırlanır.
