# Virtualizasiya Əsasları

Bu bölmə gələcəkdə aşağıdakı praktik mövzularla genişlənəcək:

- Hypervisor növləri
- Virtual maşınlar və snapshot-lar
- Storage və şəbəkə əsasları
- Lab və enterprise istifadə ssenariləri

Tam dərs hələ hazırlanır.
