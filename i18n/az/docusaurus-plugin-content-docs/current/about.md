# Haqqında

InfoSec Lessons kiber təhlükəsizlik və IT əsasları üçün iki dilli sənədləşdirmə saytıdır. Məqsəd sadədir: mövzuları elə izah etmək ki, oxuyan şəxs onları real işdə tətbiq edə bilsin, sadəcə termin əzbərləməsin.

## Bu Sayt Kim Üçündür

- Ümumi IT-dən təhlükəsizlik sahəsinə keçənlər üçün.
- Bazasını gücləndirmək istəyən junior helpdesk və sistem administratorları üçün.
- Qısa və praktik istinad materialı axtaran blue team və red team öyrənənləri üçün.
- Vaxt və pul xərcləməzdən əvvəl sertifikat yolunu müqayisə edənlər üçün.

## Burada Nə Gözləmək Olar

- Uzadılmış marketinq mətni yox, qısa və birbaşa dərslər.
- Windows, şəbəkə, server və əməliyyat təhlükəsizliyi mövzularına praktik fokus.
- English və Azərbaycan dili məzmununun mümkün qədər paralel saxlanması.
- Tək-tək bloq yazıları deyil, mərhələli böyüyən struktur.

## Redaksiya Prinsipləri

- Hype yox, dəqiqlik.
- Nizamsız nəzəriyyə yığını yox, praktik ardıcıllıq.
- Aydın scope: bir səhifə yalnız girişdirsə, bu açıq görünməlidir.
- Yeniləmələr səhifəni sadəcə uzatmamalı, istifadəni asanlaşdırmalıdır.

## Məzmun Necə Saxlanılır

- Əsas sənədlər `docs/` altında yerləşir.
- Azərbaycan dilində tərcümələr `i18n/az/docusaurus-plugin-content-docs/current/` altında eyni strukturu izləyir.
- Slug-lar EN və AZ səhifələrin rahat uyğunlaşdırılması üçün mümkün qədər sabit saxlanılır.
- Zamana həssas bələdçilərdə konkret tarixlər göstərilməli və vendor dəyişiklikləri olanda yenilənməlidir.

## Töhfə Prosesi

Saytı yaxşılaşdırmaq istəyirsinizsə:

1. Əvvəl GitHub-da issue açın və ya mövcud səhifədən başlayın.
2. Dəyişiklik konkret olsun: fakt düzəldin, strukturu yaxşılaşdırın, çatmayan praktik detal əlavə edin və ya açıq şəkildə yarımçıq mövzunu genişləndirin.
3. Yeni dərs əlavə edərkən mövcud qovluq strukturunu və EN/AZ paralelliyini qoruyun.

Saytı lokalda işə salmaq üçün:

```bash
git clone https://github.com/farzaliyevemil/infoseclessons.git
cd infoseclessons
npm install
npm start
```
