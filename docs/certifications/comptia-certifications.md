# CompTIA Certifications

This guide reflects widely referenced **CompTIA certifications as of March 23, 2026**. CompTIA updates branding, renewal rules, and portfolio structure over time, so always verify the latest certification details on the official CompTIA site.

---

## 🧭 How CompTIA Fits Into an IT Career

CompTIA is most useful when you want:

- vendor-neutral fundamentals
- a clear early-career roadmap
- practical certifications for support, networking, cloud, and cybersecurity

Unlike vendor-specific paths, CompTIA is often used to build broad baseline skills before specializing.

---

## 📚 Common Certification Groups

### Entry / Support

| Certification | Focus |
| --- | --- |
| **Tech+** | Introductory tech fundamentals |
| **A+** | End-user support, hardware, software, troubleshooting |

### Infrastructure / Operations

| Certification | Focus |
| --- | --- |
| **Network+** | Networking fundamentals, operations, troubleshooting |
| **Server+** | Server hardware, administration, operations |
| **Linux+** | Linux administration and troubleshooting |
| **Cloud+** | Multi-cloud operations, deployment, and security |

### Cybersecurity

| Certification | Focus |
| --- | --- |
| **Security+** | Core security baseline |
| **CySA+** | Detection, analysis, incident response, defensive operations |
| **PenTest+** | Practical penetration testing and assessment |
| **SecurityX** | Expert-level security architecture and engineering |

### Data / Project Adjacent

| Certification | Focus |
| --- | --- |
| **Data+** | Data analysis fundamentals |
| **Project+** | Project coordination and delivery fundamentals |

---

## 🔐 Important 2025+ Update

CompTIA **SecurityX** is the current advanced cybersecurity certification that replaced the older **CASP+** branding. If you still see CASP+ in older roadmaps, treat that as the previous name, not the current direction.

---

## 🛣️ Simple Career Paths

### Helpdesk / IT Support

`A+` -> `Network+` -> `Security+`

### Systems / Infrastructure

`A+` -> `Network+` -> `Server+` or `Linux+` -> `Cloud+`

### Cybersecurity

`A+` or `Network+` -> `Security+` -> `CySA+` or `PenTest+` -> `SecurityX`

### Cloud / Platform Operations

`Network+` -> `Linux+` or `Server+` -> `Cloud+`

---

## 🧱 Stackables Matter

One of CompTIA's practical strengths is its **stackable certification** model.

Examples from CompTIA's current pathway structure:

- `A+` + `Network+` -> **CompTIA IT Operations Specialist**
- `A+` + `Network+` + `Security+` -> **CompTIA Secure Infrastructure Specialist**
- `Security+` + `CySA+` -> **CompTIA Security Analytics Professional**
- `Security+` + `PenTest+` + `CySA+` -> **CompTIA Network Security Professional**

This makes CompTIA useful as a progression framework, not just as isolated exams.

---

## 🔄 Renewal Basics

CompTIA uses a **Continuing Education (CE)** model for many certifications.

Important points:

- Higher-level certifications can renew certain lower-level certifications
- `SecurityX` can renew lower cybersecurity certifications such as `CySA+`, `PenTest+`, `Security+`, `Network+`, and `A+`
- `Security+` can renew `Network+` and `A+`
- `Network+` can renew `A+`

This makes long-term pathway planning easier if you continue upward.

---

## ✅ Where to Start

**If you are new to IT:**
Start with `A+`

**If you already work in support and want networking:**
Go to `Network+`

**If you want cybersecurity but need a baseline first:**
Start with `Security+`

**If you want blue team / SOC work:**
Go from `Security+` to `CySA+`

**If you want offensive security fundamentals:**
Go from `Security+` to `PenTest+`

**If you want a senior technical security path:**
Build toward `SecurityX`
