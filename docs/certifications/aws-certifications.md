# AWS Certifications

This guide reflects major **AWS Certifications as of March 23, 2026**. AWS updates exam names and portfolio structure over time, so always verify the latest status on the official AWS Certification site before planning an exam path.

---

## 🧭 How AWS Certifications Are Organized

AWS groups certifications by level:

| Level | Purpose |
| --- | --- |
| **Foundational** | Broad AWS knowledge and cloud basics |
| **Associate** | Role-based technical skills |
| **Professional** | Advanced architecture, operations, and engineering |
| **Specialty** | Deep expertise in a focused domain |

---

## ☁️ Foundational

| Certification | Exam | Focus |
| --- | --- | --- |
| AWS Certified Cloud Practitioner | CLF-C02 | Overall AWS Cloud knowledge |
| AWS Certified AI Practitioner | AIF-C01 | Foundational AI, ML, and generative AI knowledge |

These are the best starting points for:

- beginners to AWS
- non-technical stakeholders
- people who want cloud literacy before deeper technical study

---

## 🧰 Associate

| Certification | Exam | Focus |
| --- | --- | --- |
| AWS Certified Solutions Architect - Associate | SAA-C03 | Designing AWS solutions |
| AWS Certified Developer - Associate | DVA-C02 | Developing and maintaining AWS applications |
| AWS Certified CloudOps Engineer - Associate | SOA-C03 | Deployment, management, and operations on AWS |
| AWS Certified Data Engineer - Associate | DEA-C01 | Data pipelines and data stores on AWS |
| AWS Certified Machine Learning Engineer - Associate | MLA-C01 | Implementing and maintaining ML solutions |

> AWS now refers to the operations-focused associate path as **CloudOps Engineer - Associate**. Older materials may still say **SysOps Administrator - Associate**.

---

## 🏗️ Professional

| Certification | Exam | Focus |
| --- | --- | --- |
| AWS Certified Solutions Architect - Professional | SAP-C02 | Advanced architecture design |
| AWS Certified DevOps Engineer - Professional | DOP-C02 | Delivery, automation, and operations |
| AWS Certified Generative AI Developer - Professional | AIP-C01 | Designing and deploying advanced AI solutions on AWS |

These are meant for more experienced practitioners, not for first-time AWS learners.

---

## 🎯 Specialty

| Certification | Exam | Focus |
| --- | --- | --- |
| AWS Certified Advanced Networking - Specialty | ANS-C01 | Complex AWS and hybrid networking |
| AWS Certified Security - Specialty | SCS-C03 | Securing AWS workloads and data |
| AWS Certified Machine Learning - Specialty | MLS-C01 | Legacy ML specialty track |

Important note:

- AWS documentation currently lists **AWS Certified Machine Learning - Specialty** with a **last day to test of March 31, 2026**

---

## 🛣️ Simple Paths

### Beginner / General Cloud

`Cloud Practitioner` -> `Solutions Architect Associate`

### Cloud Administration / Operations

`Cloud Practitioner` -> `CloudOps Engineer Associate` -> `DevOps Engineer Professional`

### Development

`Cloud Practitioner` -> `Developer Associate` -> `DevOps Engineer Professional`

### Architecture

`Cloud Practitioner` -> `Solutions Architect Associate` -> `Solutions Architect Professional`

### Cloud Security

`Cloud Practitioner` -> `Solutions Architect Associate` or `CloudOps Engineer Associate` -> `Security Specialty`

---

## ✅ Where to Start

**If you are completely new to AWS:**
Start with `AWS Certified Cloud Practitioner`

**If you want the most flexible technical starting point:**
Go to `Solutions Architect - Associate`

**If you are an operations-focused admin:**
Go to `CloudOps Engineer - Associate`

**If you are a developer:**
Go to `Developer - Associate`

**If you want a security path:**
Build a solid associate-level AWS foundation first, then move toward `Security - Specialty`
