## Learning Paths

Pick the role you're working toward and follow the lessons in order. Every path mixes foundational, intermediate, and advanced material with a green/amber/red badge so you always know the depth.

### 🟢 Beginner IT / Helpdesk
Start with [Networking Types & Topology](/networking/network-types-and-topology) → [IP Addressing](/networking/ip-addressing) → [DHCP](/networking/dhcp) → [DNS](/networking/dns) → [Common Helpdesk Tickets](/helpdesk-basics/common-helpdesk-tickets) → [Windows Run Commands](/operating-systems/windows/run-commands).

### 🛠️ Windows / Sysadmin
[Windows Server Planning](/servers/windows-server-planning) → [Active Directory Domain Services](/servers/active-directory-domain-services) → [FSMO Roles](/servers/fsmo) → [Group Policy](/servers/group-policy) → [Microsoft LAPS](/servers/laps) → [BitLocker](/operating-systems/windows/bitlocker) → [Backup](/servers/backup) → [Resilience and HA](/servers/resilience-and-availability).

### 🔵 Blue Team / SOC Analyst
[CIA Triad](/general-security/cia-triad) → [Log Analysis](/blue-teaming/log-analysis) → [Endpoint Security](/blue-teaming/endpoint-security) → [Mitigation Techniques](/blue-teaming/mitigation-techniques) → [Investigation and Mitigation](/blue-teaming/investigation-and-mitigation) → [Open-Source SIEM](/general-security/open-source-tools/siem-and-monitoring) → [Threat Intelligence and Malware](/general-security/open-source-tools/threat-intel-and-malware) → [Digital Forensics](/blue-teaming/digital-forensics).

### 🔴 Red Team / Pentester
[Networking Foundation](/networking/network-types-and-topology) → [Linux Basic Commands](/operating-systems/linux/basic-commands) → [WSL](/operating-systems/windows/wsl) → [Initial Access](/red-teaming/initial-access) → [Threat Vectors and Attack Surfaces](/red-teaming/threat-vectors-and-attack-surfaces) → [Social Engineering](/red-teaming/social-engineering) → [OWASP Top 10](/red-teaming/owasp-top-10) → [Network Attacks](/red-teaming/network-attacks) → [Malware Types](/red-teaming/malware-types) → [Attack Indicators](/red-teaming/attack-indicators) → [Penetration Testing](/red-teaming/penetration-testing) → [Threat Actors and Intel](/red-teaming/threat-actors-and-intel) → [Red-Team Tools (Open-Source)](/general-security/open-source-tools/red-team-tools).

### ☁️ Cloud Security Engineer
[Networking Foundation](/networking/network-types-and-topology) → [Secure Network Design](/networking/secure-network-design) → [Cloud Computing Security](/general-security/cloud-computing-security) → [Cloud Security Solutions](/general-security/cloud-security-solutions) → [Enterprise Security Architecture](/general-security/enterprise-security-architecture) → [Open-Source Cloud Tools](/general-security/open-source-tools/overview) → [Secrets Management and PAM](/general-security/open-source-tools/secrets-and-pam).

### 🔍 DFIR / Forensics Analyst
[Endpoint Security](/blue-teaming/endpoint-security) → [Investigation and Mitigation](/blue-teaming/investigation-and-mitigation) → [Digital Forensics](/blue-teaming/digital-forensics) → [Malware Types](/red-teaming/malware-types) → [Attack Indicators](/red-teaming/attack-indicators) → [Threat Actors and Intel](/red-teaming/threat-actors-and-intel) → [Open-Source Threat Intel and Malware](/general-security/open-source-tools/threat-intel-and-malware).

### 🛡️ GRC / Compliance Officer
[CIA Triad](/general-security/cia-triad) → [Risk and Privacy](/grc/risk-and-privacy) → [Security Policies](/grc/policies) → [Security Controls](/grc/security-controls) → [Security Governance](/grc/security-governance) → [GRC Tools (Open-Source)](/general-security/open-source-tools/grc-tools) → [IAM and Account Management](/general-security/iam-account-management).

### 🧠 Threat Intel Analyst
[Threat Actors and Intel](/red-teaming/threat-actors-and-intel) → [Attack Indicators](/red-teaming/attack-indicators) → [Malware Types](/red-teaming/malware-types) → [Threat Vectors and Attack Surfaces](/red-teaming/threat-vectors-and-attack-surfaces) → [Network Attacks](/red-teaming/network-attacks) → [Open-Source Threat Intel](/general-security/open-source-tools/threat-intel-and-malware).
