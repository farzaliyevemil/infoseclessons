This glossary is a fast lookup table for the alphabet soup of cybersecurity. Every row gives you the acronym, what it stands for, and a one-sentence definition. Where a lesson on this site covers the term in depth, the definition links straight to it.

Use it as a companion to the lessons: when a TLA (three-letter acronym) shows up mid-paragraph, jump here, then jump back. The list is grouped by first letter (A through Z) plus a small set of numeric and special entries at the end.

## Numeric and special

| Acronym | Full form | Definition |
|---------|-----------|------------|
| `0-day` | Zero-day vulnerability | A flaw exploited before a vendor patch exists. See [Vulnerability Management](/general-security/vulnerability-management). |
| `3-2-1` | 3-2-1 backup rule | Keep 3 copies of data on 2 media with 1 off-site. See [Backup](/servers/backup). |
| `802.1X` | IEEE 802.1X | Port-based network access control using EAP. See [Wireless Security](/networking/wireless-security). |
| `802.11` | IEEE 802.11 | The Wi-Fi family of wireless LAN standards. See [Wireless Security](/networking/wireless-security). |
