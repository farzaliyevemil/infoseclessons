# Microsoft Certifications

This guide reflects commonly referenced, active Microsoft certifications as of **March 23, 2026**. Microsoft updates credentials regularly, so always verify the latest exam and certification status on Microsoft Learn before booking an exam.

---

## How Microsoft Credentials Are Structured

Microsoft credentials are now broader than certifications alone:

| Type | What it means |
| --- | --- |
| **Fundamentals** | Beginner-level credentials for people new to a product area or role |
| **Associate** | Role-based certifications for day-to-day administrators, engineers, and analysts |
| **Expert** | Advanced certifications for experienced professionals and solution designers |
| **Specialty** | Focused certifications for narrower product areas |
| **Applied Skills** | Scenario-based credentials that validate hands-on tasks; these are not the same as Certifications |

> **Important:** Fundamentals certifications do **not** expire. Most Associate, Expert, and Specialty certifications are valid for **12 months** and must be renewed. Applied Skills credentials do **not** expire.

## Renewal Rules

- Fundamentals certifications do **not** expire.
- Associate, Expert, and Specialty certifications usually expire after **12 months**.
- Renewal is **free** and done through a shorter online assessment on Microsoft Learn.
- The renewal window opens **6 months before** expiration.
- If a certification fully expires, you generally need to earn it again instead of renewing it.
- Applied Skills credentials do **not** expire.

## Where Should You Start?

**If you are new to IT or cloud:**
`AZ-900` is the simplest starting point.

**If you want a Microsoft security path:**
`SC-900` -> `AZ-500` or `SC-200` / `SC-300` -> `SC-100`

**If you want a Windows Server / system administrator path:**
`AZ-900` -> `AZ-800` + `AZ-801`

**If you want a Microsoft 365 administrator path:**
`MS-900` -> `MD-102` or `MS-700` -> `MS-102`

**If you want an AI or data path:**
`AI-900` or `DP-900` -> `AI-102`, `DP-100`, `DP-300`, or `DP-600`
