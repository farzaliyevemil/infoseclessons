🧩 Open-Source Solutions

Open-source security stack — firewall/IDS/WAF, SIEM, vulnerability scanning, IAM/MFA, secrets, email security, backup, threat intelligence, red team and GRC tooling for budget-conscious teams.

📄️ Open-Source Security Stack — Overview

Roadmap for building a budget-friendly open-source security stack across firewall/IDS, SIEM, vulnerability scanning, IAM, secrets, email, backup, threat intel, red team and GRC — plus IT asset management as the foundation.

📄️ Open-Source Firewall, IDS/IPS, WAF and NAC

Open-source perimeter and inline defense — OPNsense and pfSense (firewalls), Suricata, Zeek and Snort (IDS/IPS), ModSecurity and BunkerWeb (WAF), and PacketFence (NAC).

📄️ Open-Source SIEM, Logging and Monitoring

Open-source detection and observability stack — Wazuh and OSSEC for HIDS, the ELK / OpenSearch stack for log aggregation, and Zabbix / Prometheus / Uptime Kuma for infrastructure monitoring.

📄️ Open-Source Vulnerability Scanning and Application Security

Open-source vulnerability assessment and application security tooling — OpenVAS for infrastructure scanning, OWASP ZAP and Nuclei for DAST, Semgrep and Bandit for SAST, Trivy and Syft for SBOM and container scanning.

📄️ Open-Source IAM and MFA

Open-source identity and access management — Keycloak as the de-facto enterprise SSO, Authentik as the modern UI-first alternative, the Ory stack for headless IAM, plus Authelia and PrivacyIDEA for MFA front-ends.

📄️ Open-Source Secrets Management and Privileged Access

Open-source secrets management (HashiCorp Vault, Infisical, Conjur), privileged access (Teleport, JumpServer, Pritunl Zero) and password vaults (Vaultwarden, Passbolt, Psono) — the three layers of secret-handling for humans, machines, and admins.

📄️ Open-Source Email Security

Open-source mail security stack — Rspamd and SpamAssassin for filtering, Proxmox Mail Gateway for scanning, and Mailcow / Mail-in-a-Box / iRedMail as full self-hosted mail server bundles.

📄️ Open-Source Backup and File Sharing

Open-source backup and storage stack — BorgBackup and Restic for deduplicated backups, UrBackup for image-based, plus Nextcloud, Seafile and OnionShare for file sharing and self-hosted cloud.

📄️ Open-Source Threat Intelligence and Malware Analysis

Open-source threat intel platforms (OpenCTI, MISP, YETI) and malware analysis sandboxes (Cuckoo, CAPEv2, IntelOwl) — the tools that turn raw indicators into operational intelligence.

📄️ Open-Source Red Team and Adversary Emulation

Open-source tools for offensive security testing — GoPhish for phishing simulation, MITRE Caldera and Atomic Red Team for adversary emulation, Sliver as a modern C2, plus APTSimulator and RTA for technique simulation.

📄️ Open-Source GRC Tools

Open-source governance, risk and compliance tooling — CISO Assistant CE, Eramba CE, OpenGRC and Gapps for risk register, control mapping, audit evidence, and policy lifecycle management.
