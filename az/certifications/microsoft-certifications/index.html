Microsoft Sertifikatları

Bu bələdçi 23 mart 2026 tarixinə aktiv və geniş istifadə olunan Microsoft sertifikatlarını əhatə edir. Microsoft credential portfelini mütəmadi yenilədiyi üçün imtahana qeydiyyatdan əvvəl son vəziyyəti mütləq Microsoft Learn üzərindən yoxlayın.

Microsoft Credential Sistemi Necə Qurulub?​

Microsoft-da artıq təkcə sertifikatlar yox, daha geniş credential sistemi var:

NövMənası
FundamentalsMəhsul və ya rol üzrə başlanğıc səviyyəli credential
AssociateGündəlik admin, engineer və analyst rolları üçün rol əsaslı sertifikat
ExpertTəcrübəli mütəxəssislər və solution designer-lər üçün daha yüksək səviyyə
SpecialtyDaha dar fokuslu texnologiyalar üçün sertifikat
Applied SkillsPraktik ssenari əsaslı credential; bunlar Certification ilə eyni deyil

Vacib qeyd: Fundamentals sertifikatlarının müddəti bitmir. Associate, Expert və Specialty sertifikatlarının çoxu 12 ay keçərlidir və yenilənməlidir. Applied Skills credential-larının müddəti bitmir.

Renewal Qaydaları​

  • Fundamentals sertifikatlarının müddəti bitmir.
  • Associate, Expert və Specialty sertifikatları adətən 12 ay keçərlidir.
  • Renewal ödənişsizdir və Microsoft Learn üzərindən qısa onlayn assessment ilə edilir.
  • Renewal pəncərəsi expiration tarixindən 6 ay əvvəl açılır.
  • Sertifikat tam expire olduqdan sonra adətən renewal olmur; credential yenidən earn edilməlidir.
  • Applied Skills credential-larının müddəti bitmir.

Haradan Başlamaq Daha Məntiqlidir?​

Əgər IT və ya cloud-a yenisənsə: AZ-900 ən rahat başlanğıc nöqtəsidir.

Əgər Microsoft security istiqamətinə getmək istəyirsənsə: SC-900 -> AZ-500 və ya SC-200 / SC-300 -> SC-100

Əgər Windows Server / sistem administratoru olmaq istəyirsənsə: AZ-900 -> AZ-800 + AZ-801

Əgər Microsoft 365 administratoru olmaq istəyirsənsə: MS-900 -> MD-102 və ya MS-700 -> MS-102

Əgər AI və ya data istiqamətinə getmək istəyirsənsə: AI-900 və ya DP-900 -> AI-102, DP-100, DP-300 və ya DP-600
