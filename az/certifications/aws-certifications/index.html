AWS Sertifikatları

Bu bələdçi 23 mart 2026 tarixinə əsas AWS sertifikatlarını əhatə edir. AWS exam adlarını və portfel strukturunu zaman-zaman dəyişdirdiyi üçün imtahan planlamazdan əvvəl son vəziyyəti rəsmi AWS Certification səhifəsində yoxlamaq lazımdır.

🧭 AWS sertifikatları necə qurulub?​

AWS sertifikatları səviyyələrə bölür:

SəviyyəMəqsəd
FoundationalGeniş AWS biliyi və cloud əsasları
AssociateRol əsaslı texniki bacarıqlar
ProfessionalDaha dərin arxitektura, əməliyyat və engineering
SpecialtyDar sahədə yüksək səviyyəli ekspertiza

☁️ Foundational​

SertifikatİmtahanFokus
AWS Certified Cloud PractitionerCLF-C02Ümumi AWS Cloud biliyi
AWS Certified AI PractitionerAIF-C01Əsas AI, ML və generative AI anlayışları

Bunlar daha uyğundur:

  • AWS-ə yeni başlayanlar
  • texniki olmayan stakeholder-lər
  • dərin texniki yola keçməzdən əvvəl cloud literacy istəyənlər

🧰 Associate​

SertifikatİmtahanFokus
AWS Certified Solutions Architect - AssociateSAA-C03AWS həllərinin dizaynı
AWS Certified Developer - AssociateDVA-C02AWS tətbiqlərinin hazırlanması və saxlanması
AWS Certified CloudOps Engineer - AssociateSOA-C03Deployment, idarəetmə və əməliyyatlar
AWS Certified Data Engineer - AssociateDEA-C01Data pipeline və data store-lar
AWS Certified Machine Learning Engineer - AssociateMLA-C01ML həllərinin qurulması və saxlanması

Köhnə materiallarda operations associate yolu hələ də SysOps Administrator - Associate kimi görünə bilər. Cari ad CloudOps Engineer - Associate-dir.

🏗️ Professional​

SertifikatİmtahanFokus
AWS Certified Solutions Architect - ProfessionalSAP-C02Advanced architecture design
AWS Certified DevOps Engineer - ProfessionalDOP-C02Delivery, automation və operations
AWS Certified Generative AI Developer - ProfessionalAIP-C01AWS üzərində advanced AI həllərinin dizaynı və deployment-u

Bu səviyyə ilk AWS imtahanı üçün yox, daha təcrübəli practitioner-lər üçündür.

🎯 Specialty​

SertifikatİmtahanFokus
AWS Certified Advanced Networking - SpecialtyANS-C01Mürəkkəb AWS və hybrid şəbəkə arxitekturası
AWS Certified Security - SpecialtySCS-C03AWS workload və məlumat təhlükəsizliyi
AWS Certified Machine Learning - SpecialtyMLS-C01Köhnə ML specialty istiqaməti

Vacib qeyd:

  • AWS sənədlərində AWS Certified Machine Learning - Specialty üçün 31 mart 2026 son imtahan tarixi göstərilir

🛣️ Sadə karyera yolları​

Başlanğıc / ümumi cloud​

Cloud Practitioner -> Solutions Architect Associate

Cloud administration / operations​

Cloud Practitioner -> CloudOps Engineer Associate -> DevOps Engineer Professional

Development​

Cloud Practitioner -> Developer Associate -> DevOps Engineer Professional

Architecture​

Cloud Practitioner -> Solutions Architect Associate -> Solutions Architect Professional

Cloud security​

Cloud Practitioner -> Solutions Architect Associate və ya CloudOps Engineer Associate -> Security Specialty

✅ Haradan başlamaq daha düzgündür?​

Əgər AWS-ə tam yenisənsə: AWS Certified Cloud Practitioner

Əgər ən çevik texniki başlanğıc istəyirsənsə: Solutions Architect - Associate

Əgər operations/admin yönümlüsənsə: CloudOps Engineer - Associate

Əgər developer-sənsə: Developer - Associate

Əgər security istiqaməti istəyirsənsə: Əvvəl associate səviyyəsində güclü AWS bazası qur, sonra Security - Specialty-ə keç
